The app uses ReactNative framework, 'react-navigation' library for navigating through the pages, has functionality of browsing through the different categories of movies and viewing details about each movie including it's rating, description and trailer.
Using a promise-based HTTP client Axios the application is getting data from themoviedb.org API and displaying it for the user. The user then selects an item and a new query to the API is made to fetch more details about a specific movie of the genre. Additionally the user can click the Search button and search for any movie in the database, then the application gets all the movies that match the query and displays them for the user. Again he can select to view the details about each result.
The trailer is embedded into the page using 'react-native-youtube-iframe' library which takes the Youtube id from the api and displays the video alongside the rating and the description of the selected movie.
