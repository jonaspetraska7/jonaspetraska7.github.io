Login page designed by me. A simplistic logo, two color scheme throughout the whole app. The login system uses FireBase database. Supports logging in from Facebook.
We built a custom messenger in the app. It uses FireBase's Real-time database to quickly send and receive messages between users and display them the app. In this application we made it so the users can only contact their liked event's creator and ask them details or arrange a meeting.
The App can create events, modify them, detect the events you have liked and then connect you with the owner of the event to discuss the details or the pricing. The navigation of the page is made in Android studio with native components.
The app offers the famous Tinder-like swiping cards which register the users event preferences and lets them discover new events posted by others, see their own event in action and helps them connect with the event publishers.
